# UI Redesign

Use this skill when improving layout, spacing, and visual clarity.

## Rules
- Do NOT change business logic
- Focus only on UI, layout, and styling
- Keep design minimal and clean
- Improve spacing and hierarchy
- Prioritize mobile usability
- Keep black/red aesthetic

## UX Focus
- Active set must stand out
- Completed sets should be compact
- Reduce clutter
- Optimize for fast interaction

## Workflow
1. Analyze current UI
2. Identify clutter or friction
3. Propose minimal UI improvement
4. Implement without breaking logic
